# Lesson 1.5
This prosject is created to learn Git, ESLint, Vite, Prettier, Readme, PR, Github Actions

## Avaiable npm scripts
In the project directory you can run:
- `npm run dev` - Start development server (Vite)
- `npm run build` - Build for production
- `npm run test` - Run tests
- `npm run lint` - Run ESLint
- `npm run preview` - Preview production build

## Enviroment Variables
Create a .env file in the root directory:
```
VITE_API_URL=https://jsonplaceholder.typicode.com/posts
```
Restart the devolpment server after modifing the env. file
The varibale is accessed in the app using: 
```
import.meta.env.VITE_API_URL
```
